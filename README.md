🗳️ votacao_RTorres
Aplicativo de votação online desenvolvido para os cooperados da RTorres, com foco em experiência do usuário (UX), garantindo praticidade, segurança e acessibilidade no processo de decisão coletiva.

✨ Objetivo
O sistema foi criado para permitir que os cooperados participem de votações de forma rápida, transparente e segura, diretamente pelo navegador, sem necessidade de instalação de aplicativos nativos.
Um dos principais diferenciais é o trabalho de UX, que buscou tornar a interface simples, intuitiva e agradável, reduzindo barreiras tecnológicas para todos os usuários.

🔧 Tecnologias Utilizadas
Flutter Web → construção da interface responsiva e moderna

MobX + Modular → gerenciamento de estado e arquitetura escalável

Geolocator → validação de localização dos usuários

Lottie → animações leves e interativas

Autenticação via SMS → segurança no acesso ao sistema

UI responsiva → adaptada para diferentes dispositivos e tamanhos de tela

🚀 Como rodar o projeto
Instale as dependências:

bash
flutter pub get
Execute em ambiente web (Chrome recomendado):

bash
flutter run -d chrome
