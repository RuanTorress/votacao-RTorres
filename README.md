# 🗳️ votacao_uniodonto

Sistema de votação online para cooperados da Uniodonto.

## 🔧 Tecnologias Utilizadas

- Flutter Web
- MobX + Modular
- Geolocator (Localização)
- Animações com Lottie
- Autenticação via SMS
- UI moderna e responsiva
